# evg85-start-template

Базовый стартовый шаблон для веб-разработки


[preprocess]: https://github.com/jsoverson/preprocess

## 2 режима:
- dev (режим development)
  - html -> html (с прогонкой через preprocess)
  - sass -> css (не сжатый, с подключением sourcemaps)
- prod (режим production)
  - html -> html (с прогонкой через preprocess)
  - sass -> css (сжатый без soursemaps)
